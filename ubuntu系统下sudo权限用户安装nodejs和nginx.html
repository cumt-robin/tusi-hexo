ubuntu系统下sudo权限用户安装nodejs和nginx

为了支撑公司某 ios app 上线，今天做了个隐私政策 h5 页面并上线，顺手体验了一把 ubuntu系统的 sudo 权限。本来想用 nodejs 的 express 框架搭个简单的静态资源托管服务，然后用 nginx 做下反向代理。但是在安装 express-generator 时遇到点问题，可能 sudo 权限玩得不够熟练，跟 root 用户还是有很大区别的。本文简单说下自己在 ubuntu 系统下 sudo 权限用户安装 nodejs 和 nginx 的过程。

安装后要知道的几点是：

  • 配置文件所在目录

其他的坑
